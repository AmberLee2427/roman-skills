# Style Profiles

## Shared Baseline
- Must include vector export (`pdf` or `svg`).
- Must include raster export (`png`) with `dpi >= 300`.
- Axes labels should include units where physically meaningful.

## APJ
- TeX-enabled text strongly preferred.
- Serif font family preferred.

## MNRAS
- TeX-enabled text preferred.
- Grayscale distinguishability required for multi-series figures.

## A&A
- Vector export required.
- Concise axis labeling with units required.

# Style Profiles

Canonical structured profile data now lives in `../assets/journal_profiles.json`.
This reference is a human-readable summary of the profile intent.

## Shared Baseline
- Must include vector export (`pdf` or `svg`).
- Must include raster export (`png`) with `dpi >= 300`.
- Axes labels should include units where physically meaningful.

## APJ
- TeX-enabled text strongly preferred.
- Serif font family preferred.
- Shared project defaults target single-column or double-column astrophysical journal widths.

## Nature
- Sans-serif labeling preferred.
- Single-column and double-column widths follow Nature production guidance.
- Keep figure typography compact and avoid oversized text.

## Nature Initial Submission
- Use referee-quality figures at 300 dpi or higher.
- Preferred widths are 90 mm (single column) and 180 mm (double column), with max height 170 mm.
- Use Arial or Helvetica where possible; keep text in the 5-7 pt range at final size.

## Nature Revised / Production
- Use editable vector text with standard sans-serif fonts.
- Keep text in the 5-7 pt range; panel labels should be 8 pt bold lowercase.
- Avoid background gridlines and coloured text.
- Bitmap images should be 450 dpi or above when supplied for production.

## Science
- Sans-serif labeling preferred (Helvetica if available).
- Figure title belongs in the caption, not in the figure body.
- Single-column and double-column widths follow Science preparation guidance.

## MNRAS
- TeX-enabled text preferred.
- Grayscale distinguishability required for multi-series figures.

## A&A
- Vector export required.
- Concise axis labeling with units required.
